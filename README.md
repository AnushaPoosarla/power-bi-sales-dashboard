# power-bi-sales-dashboard

Project Overview:

This project demonstrates an end-to-end Power BI dashboard built using a messy sales dataset. The goal was to clean, transform, and visualize the data to generate meaningful business insights.

Data cleaning and transformation using Power Query:

Created DAX measures:
Total Sales
Total Orders
Total Quantity
Average Selling Price
Built interactive dashboard with:
Sales by Category
Sales by Region
Top Performing Products
Monthly Sales Trend
Implemented Top N filtering
Added slicers (City, Category) for dynamic filtering
